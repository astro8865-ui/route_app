# ルート作成ツール (Route Planner)

地図上でクリックしてルートを作成・保存・結合できるWebアプリです。OSM (OpenStreetMap) と OSRM を利用しています。

## 🚀 公開URL
PC
https://astro8865.github.io/route_app/index.html

Smart Phone
https://astro8865.github.io/route_app/indexsp.html

## 🛠 機能
- **ルート作成**: 地図をクリックしてポイントを打つと自動でルートが描画されます。
- **ポイント編集**: ドラッグ＆ドロップでポイントを微調整できます。
- **保存機能**: 作成したルートをブラウザのローカルストレージに保存できます。
- **読み込み・結合**:
    - **読込**: 既存ルートを一度クリアしてから、保存済みルートを読み込みます。
    - **結合**: 現在のルートの末尾に、保存済みルートを連結します。
- **現在地表示**: ワンクリックで現在の位置へ移動します。

## 📝 使い方
1. 地図上をクリックしてポイントを作成します。
2. 作成したルートは「ルート保存」ボタンから名前を付けて保存できます。
3. 「保存ルート一覧」から、ルートを読み込んだり、現在のルートに結合したりできます。
